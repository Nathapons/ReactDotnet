
# Full Stack: React + ASP.net



## Programing Language

 - React

 - ASP.net

 - Docker


## Clone Project
Clone the project

```bash
  git clone https://github.com/Nathapons/ReactDotnet
  cd ReactDotNet
```



## Frontend Run Locally

Go to the project directory

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```


## Backend Run Locally

Go to the project directory

```bash
  cd backend
```

Install dependencies

```bash
  dotnet restore
```

Start the server

```bash
  dotnet run
```
